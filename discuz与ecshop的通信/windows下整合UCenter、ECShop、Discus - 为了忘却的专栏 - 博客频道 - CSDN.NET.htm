windows下整合UCenter、ECShop、Discus

分类： 为了忘却 2011-01-05 14:42 1871人阅读 评论(0) 收藏 举报
windowsphpmyadminupload数据库浏览器mysql

安装环境：windows

WampServer 2.1c [28/12/10]
 Inclus :
 - Apache 2.2.17
 - Php 5.3.4 (version 64 bits)
 - Php 5.3.3 (version 32 bits)
 - Mysql 5.1.53 (version 64 bits)
 - Mysql 5.5.8 (version 32 bits)
 - XDebug 2.1.0-5.3-vc6 (32 bits)
 - XDebug 2.1.0-5.3-vc9 (64 bits)
 - XDC 1.5
 - PhpMyadmin 3.2.0.1
 - SQLBuddy 1.3.2
 - webGrind 1.0

1.       下载UCenter
下载地址：http://www.comsenz.com/downloads/install。选择的是UTF-8的

2.       解压UCenter_1.5.2_SC_UTF8文件，可以直接将upload文件夹直接复制到项目的文件夹里。当然项目要布置到服务器上。

3.       安装UCenter

打开phpmyadmin的服务，在浏览器上输入http://localhost/upload/install 然后就出现一个安装向导，在填写数据库名时，由于前面phpmyadmin在xp下安装的有一点问题，数据库名还是写的root 密码填空。一路到最后就安装好了。完成之后需要将安装文件删除，及删除intall文件夹。（本次安装后，部分文件夹会报错，页面还是可以出来，尝试着改了一下，没有出现问题）

4.       下载ECSHOP软件包

下载地址：http://www.ecshop.com 。选择的是UTF-8的

5.       解压缩ECSHOP软件包，将ECShop_V2.7.2_UTF8_Release0604解压，在项目里新建一个文件夹ecshop，将upload文件夹下的文件复制到新建的ecshop文件夹下。

6.       安装ecshop

打开phpmyadmin的服务，在浏览器上输入http://localhost/ecshop/install 然后就出现一个安装向导，跟着一路下来，能够登陆进去就安装成功了。成功后也要把里面的install安装文件删除。

7.       整合uCenter。

整合之前，将UCenter_1.5.2_SC_UTF8 /advanced文件的uc_client文件复制到ecshop文件夹下。登录ecshop，http://localhost/ecshop/admin/ 找到菜单>会员管理>会员整合，找到UCenter，点安装。URL填UCenter的目录http://localhost/upload 。然后跟着向导一路下去。成功后会有提示。打开UCenter的界面，会发现数据库里的数据多了几条。那是ecshop的数据整合到ucenter上了。

8.       整合discuz

下载Discuz_X1.5_SC_UTF8 ，地址：http://download.comsenz.com/DiscuzX/ ，将upload文件夹下的内容复制到项目里的discuz文件夹里，可以像上面一样安装，也可以登录UCenter用户管理中心>应用管理，添加新的应用，url也是和上面的一样，到install文件。安装完后也应把install文件删除。

 

  这样整合之后，所有的数据库都交到UCenter来管理。打开各自的页面就能够进行丰富的设置。
